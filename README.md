# ARSCII_ART_web

사진을 넣으면 아스키 아트를 만들어 주는 웹

사진을 넣으면 픽셀 마다 rgb값을 추출한다.
0.299R + 0.587G + 0.114B 수식을 거쳐 픽셀의 밝기를 구한다.
밝기값에 맞는 미리 선택한 문자를 해당칸에 입력한다.
[@$#Y!=+~-ㅤ](0~24/25~49/50~74/이런식으로...)
ex)
rgb(56, 50, 53) >> 0.299R + 0.587G + 0.114B >> 52 >> [#]

다른사람이 만든 아스키 아트를 쉽게 복사 가능함.

게시글을 들어가면 복사 버튼이 따로 있음(클립보드에 복사해줌)

각각의 아스키 아트들을 하나의 게시글로 취급하는 커뮤니티 사이트

하나의 게시글에 댓글을 달 수 있음.

유저는 익명과 닉네임으로 나뉨.

별도의 닉네임 입력 없이 댓글을 입력하면 게시글에서 이 댓글이 쓰인 순서애 따라 익영N으로 표시됌.
닉네임을 입력하고 작성시 입력된 닉네임이 표시됌.
